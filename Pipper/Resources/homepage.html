Welcome!

Just a webview that floats over other apps, useful for watching YouTube videos while coding and stuff.

It's still just a proof of concept, you can find it on GitHub

Usage

  • Search using the icon below
  • Search from clipboard using Cmd + Shift + V
  • Search Engine can be configured in settings
  • Window is resizable, some predefined sizes are available in settings
  • User agent can be changed from settings
  • App will start at its latest location and size
